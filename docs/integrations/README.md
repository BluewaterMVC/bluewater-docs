## 📘 Overview

This section documents how Bluewater interacts with **external services and platforms**—such as cloud providers, messaging queues, monitoring systems, and third-party APIs.

Each subdirectory within `docs/integrations/` focuses on a specific integration category or vendor ecosystem.

- `/monitoring/` – external observability tools (e.g., Sentry, Datadog)
- `/queues/` – brokers like RabbitMQ, Beanstalkd, or Redis Streams
- `/secrets/` – external secret managers (e.g., Vault, AWS Secrets Manager)
- `/cloud/` – deployment and service integration for cloud platforms
- `/sdks/` – consuming and exposing APIs to SDK consumers
